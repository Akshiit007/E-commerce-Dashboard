# E-commerce-Dashboard
📌 Short Description
An interactive Power BI dashboard developed to analyze and visualize e-commerce sales data. The dashboard provides insights into sales performance, profitability, customer purchasing patterns, and product category trends through dynamic visualizations and filters. It enables businesses to monitor key metrics and make data-driven decisions efficiently.

🛠️ Tech Stack
Power BI Desktop – Dashboard development and visualization
Power Query – Data cleaning and transformation
DAX (Data Analysis Expressions) – Measures and calculated columns
Data Modeling – Building relationships between tables
Excel/CSV – Source data storage

✨ Features & Highlights
📈 Business Problem
Businesses often struggle to track sales performance and identify key profit drivers due to scattered and unorganized data.
💡 Solution
Developed an interactive dashboard that consolidates sales data into meaningful visualizations, helping stakeholders understand performance trends and make informed decisions.

🔍 Key Features
Interactive Quarter-wise Filtering (Q1-Q4).
KPI Cards displaying:
Total Sales Amount
Total Profit
Total Quantity Sold
Average Order Value (AOV)
State-wise sales analysis.
Category-wise revenue distribution.
Monthly profit trend analysis.
Payment mode analysis.
Profit analysis by product sub-category.
Dynamic slicers and drill-down capabilities.

📊 Key Insights
Maharashtra recorded the highest sales among all states.
Clothing contributed the largest share of revenue.
Cash on Delivery (COD) was the most preferred payment method.
Printers generated the highest profit among all sub-categories.
Profit varied significantly across different months.
